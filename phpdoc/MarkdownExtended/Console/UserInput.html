MarkdownExtended\Console\UserInput

class UserInput

A class to manage command line options based on a set of definitions

Constants

NEGATE_SUFFIX

ARG_NULL

Use this when concerned option does NOT have any argument (i.e.

for flags)
ARG_OPTIONAL

Use this when concerned option can accept an argument but this is not required (i.e.

a default value is defined)

The optional argument MUST be written separated to the option by an equal sign.
ARG_REQUIRED

Use this when concerned option REQUIRES an argument

The equal sign between the option and its argument is not required.
TYPE_BOOL

Use this to define a boolean option (i.e.

1 or 0)

This is the default for option with no argument (i.e. self::ARG_NULL)
TYPE_STRING

Use this to define an option as a string
TYPE_PATH

Use this do define an option as a file path

An error will be thrown if the option only have this type and its value can not be found in file system.
TYPE_LISTITEM

Use this to define an option as a list item

Using this type, you are REQUIRED to define a list argument in the option's definition.
